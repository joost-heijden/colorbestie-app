# Placeholder PWA icons

These are SVG placeholders for PWA icon slots:

- `icon-192.svg` → 192x192
- `icon-512.svg` → 512x512
- `icon-maskable-512.svg` → 512x512 (maskable safe area)

For production app stores/device polish, replace these with PNG assets of the same sizes and update `app/manifest.ts` `type` and paths accordingly.
